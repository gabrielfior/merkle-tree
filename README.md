# merkle-tree

## Getting started

```
cd /path/to/merkle-tree-repo
docker compose up --build --force-recreate --remove-orphans
```

Now navigate to
- `http://localhost:5000/docs` for the server endpoints specs
- `http://localhost:8501` for the client webpage
